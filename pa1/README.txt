Simple_add.c - Contains the code for the system call simple_add.
Location: kernel/linux-4.4.0/arch/x86/kernel/simple_add.c

Makefile - A collection of all of the object files for the system calls so that they can be easily linked.
Location: kernel/linux-4.4.0/arch/x86/kernel/Makefile

Syscall_64.tbl - A table containing all of the possible  system calls.
Location: kernel/linux-4.4.0/arch/x86/entry/syscalls/syscall_64.tbl

Syscalls.h - The header file containing all of the function definitions for the system calls.
Location: kernel/linux-4.4.0/arch/x86/include/linux/syscalls.h

Syslog - A collection of event messages that are generated by the system or software that are then stored.
Location: /var/log/syslog

Test.c - A test program used to call the new system call Simple_add.
To run complile Test.c by entering "g++ test.c" in the terminal and then enter "./a.out" to run the program.
